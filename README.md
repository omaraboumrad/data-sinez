# Data Sinez

Slack stats slide builder

- Purpose: Fun
- Correctness: None
- Accuracy: None

# Requirements

- Build: [Python3](https://www.python.org/downloads/)
- Present: [mdp](https://github.com/visit1985/mdp)

```
$ pip install -r requirements.txt
```

## Run

```
$ ./build.py stats.md
$ mdp output/stats.md
```
